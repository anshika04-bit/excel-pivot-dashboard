# Excel Pivot Table Dashboard

## Project Overview
This project contains an Excel-based interactive dashboard created using
Pivot Tables, Pivot Charts, and Slicers.
The dashboard helps in analyzing data and tracking key performance metrics
in a simple and visual way.

---

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Functions (SUM, COUNT, IF, etc.)
- Data Cleaning & Formatting

---

## Dashboard Features
- Interactive Pivot Table–based analysis
- KPI cards for quick insights
- Dynamic Pivot Charts (Bar / Line / Column)
- Slicers for filtering data (Date, Category, Region, etc.)
- Clean and user-friendly layout

---

## Files in This Repository
- `Pivot_Dashboard.xlsx` → Main Excel dashboard file
- `README.md` → Project documentation

---

## How to Use
1. Download the Excel file from this repository
2. Open it in Microsoft Excel (2016 or later recommended)
3. Go to the **Dashboard** sheet
4. Use slicers to filter data and analyze insights
5. To refresh data: **Data → Refresh All**
